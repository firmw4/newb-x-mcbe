NEWB x GEMINI created by firmw4
Another variant to improve your Minecraft visual, dynamic lighting and immersive atmosphere. This variant uses visual inspiration and coloring from AI.

There are several packages, including Normal, Ultra and Lite.
Difference between Normal, Ultra and Lite:

Normal
  Default packages with wave effects, better fog, aurora and others
Ultra
  Enhanced normal version with Fancy SunMoon Godray, Twinkling Star and some upgraded features
Lite
  Remove and disable some features from the normal package, disable wave effects by default. For better performance
